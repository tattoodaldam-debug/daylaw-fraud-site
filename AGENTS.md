# 프로젝트 규칙 (데이법률사무소 금융사기 사이트)

## 작업 방식
- 각 작업 단계가 끝날 때마다 git 커밋을 하고 사용자에게 "완료"라고 보고한다.
- 사용자는 개발 비전공자이므로 각 단계에서 하는 일을 한국어 한 문장으로 설명한다.

## 콘텐츠 절대 규칙 (변호사 광고 규정 준수)
- 회사 표기는 반드시 **'데이법률사무소'**. **'법무법인'이라는 단어는 절대 사용 금지** (법무법인이 아니므로 허위 표기).
- 기존 홈페이지 day-lawfirm.com의 문장을 복사하지 않는다. 모든 문구는 새로 작성.
- 단정적/보장성 표현 절대 금지. 금지 목록: '100% 환수', '무조건 승소', '보장', '전액 회수', '무조건 해결', '무조건 무혐의', '성공률', '전국 1위', '최고 전문', '반드시 돌려받을', '즉시 회수', '불송치 보장', '합의하면 끝', '지금 안 하면 큰일', '처벌을 피할 수 있습니다'. 발견 시 "사건별로 검토합니다" 계열의 비단정 표현으로 교체한다. (예외: 사기 수법을 인용·설명하는 문맥의 '원금 보장' 등은 허용)
- 성공사례는 실제 사례만(사무소 제공분만) 게시하고, 모든 법률 콘텐츠는 게시 전 변호사 검수를 거친다.
- 사무소 기본 정보는 `materials/company-info.rtf` 참고.
